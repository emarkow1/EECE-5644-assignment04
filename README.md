# EECE-5644-assignment04

## Downloading the Dataset

The dataset [titanic.csv](titanic.csv) is present in the repository already, tracked using git lfs. However, if needed, it can be downloaded from [https://www.kaggle.com/datasets/yasserh/titanic-dataset/data](https://www.kaggle.com/datasets/yasserh/titanic-dataset/data)

## Running the Project

All the code cleaning the data, building the model, and testing it is present in the jupyter notebook [assignment04.ipynb](assignment04.ipynb). It is recommended to run the jupyter notebook and install the required dependencies in a virtual environment.

### Setting up the venv
Run in the terminal:
```
python3.14 -m venv .venv
```
### Installing Dependencies
```
pip install -r requirements.txt
```

After this, the notebook can be opened and all cells ran
